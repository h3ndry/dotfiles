## Simple dotfiles

This repo contain dotfiles of programmes I use on my machines and few personal scrips that
help me get staff done.

### Programmes
- bat
- dunst
- mpd
- Neovim
- sxhd
- zathura


### Scrips
 * `notestaker` A scripst that lanuch nvim in the mindle of the screen ready to take notes
 in markdown with time and date already pre populated. based on a youtube video I have watch.

 * `notesreader` A scripst to output latest notes using bat

 * `quickmove` A script that allow me to quickly open any project on pre-define folder on a 
nvim instance.  Motivate by this youtube video

* `snippy` A script that allow me to enter repetive data quick and easy
